Features
--------
lorawan-sk-backend is an IoT backend to archive and serve IoT information gathered from multiple sources - notably LoRaWAN and Sigfox plus some propriatery IoT gateways. The solution is written in Python using Django framework. Accepts and serves data using HTTP REST API.
